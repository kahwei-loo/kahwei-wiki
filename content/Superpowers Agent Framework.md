# Superpowers — Agent Engineering Framework

> A skills framework and development methodology for AI coding agents. Enforces disciplined engineering: understand → design → plan → test → code, instead of "generate code immediately." 40K+ stars.

## The Problem

AI coding agents write code fast, but "fast" ≠ "good":
- Skip requirements gathering
- Write code before thinking about design
- Don't write tests first
- Don't plan for edge cases

## The Superpowers Methodology

```
1. BRAINSTORM  → Understand what's actually needed
2. DESIGN      → Plan the architecture
3. PLAN        → Break into small, verifiable tasks
4. TEST FIRST  → Write tests before code (TDD)
5. IMPLEMENT   → Write code to pass tests
6. VERIFY      → Ensure everything works
```

Like giving your AI coding assistant an **engineering bootcamp**.
