# GitNexus — Code Knowledge Graph

> Browser-based tool that turns any GitHub repo into an interactive knowledge graph. Zero installation — paste a URL, get a visual map of dependencies, call chains, and execution flows, plus a built-in AI agent for code Q&A.

## How It Works

```
GitHub Repo URL → GitNexus → Interactive Knowledge Graph + AI Agent
```

Everything runs in the browser. No server, no API keys, no installation.

## Key Features

- **Zero install**: Open in browser, paste repo URL, done
- **Visual mapping**: Dependencies, call chains, clusters, execution flows
- **Built-in AI agent**: Ask questions about the codebase
- **10K+ stars** (4K earned in one week)

## Use Case

Join a new project with hundreds of files and no docs → drop the URL into GitNexus → get an interactive visual map in minutes instead of spending days reading code.
