# OpenClaw Personal AI Assistant

> A local-first, self-evolving personal AI assistant that works across 50+ messaging platforms. 247K+ stars — fastest-growing open-source project in GitHub history.

## What It Is

A personal AI assistant that runs on your own devices, connects to platforms you already use (WhatsApp, Telegram, Slack, Discord, etc.), and can write its own code to learn new skills.

## Key Differentiators

| Feature | OpenClaw | ChatGPT | Google Gemini |
|---------|----------|---------|---------------|
| Runs locally | Yes | No | No |
| Subscription | No (BYOK) | $20/mo | $20/mo |
| Multi-platform | 50+ | Web + App | Web + App |
| Self-evolving skills | Yes | No | No |
| Open source | Yes | No | No |

## Core Features

- **Platform-agnostic**: One AI across WhatsApp, Telegram, Slack, Teams, Discord, Signal, LINE, IRC, and more
- **Self-evolving**: Can write its own code to learn new capabilities on the fly
- **Voice support**: Speaks and listens on macOS, iOS, Android
- **Productivity integrations**: Apple Notes, Notion, Obsidian, Trello, GitHub, cron jobs, webhooks
- **BYOK model**: Bring your own API key from any LLM provider

## Ecosystem

- **nanobot** (by HKUDS): Ultra-lightweight OpenClaw variant
- **ClawWork**: "OpenClaw as Your AI Coworker"
- Growing community of plugin/skill developers
